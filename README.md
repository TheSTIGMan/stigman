# stigman

AI-powered DISA STIG compliance scanner for Ubuntu Server 24.04 LTS. 
The tool runs locally on the machine being scanned and uses natural language input to drive OpenSCAP scans, summarize findings, and optionally remediate issues.

## How It Works

The tool acts as a chatbot in the terminal, powered by either Anthropic (Claude) or OpenAI (GPT). The LLM orchestrates the complex OpenSCAP commands behind the scenes.

**Example Interaction**
```
$ sudo stigman

Welcome to stigman! AI-powered DISA STIG compliance scanner.

First-time setup detected.
Select your LLM provider:
  [1] Anthropic (Claude)
  [2] OpenAI (GPT)
Choice: 1
Enter your Anthropic API key: sk-ant-...
Config saved to ~/.stigman/config.json

You're all set! Type a command or ask a question.

> scan this machine for STIG compliance

Detecting OS... Ubuntu 24.04 LTS confirmed.
Checking prerequisites... openscap-scanner and ssg-ubuntu2404 are installed.
Running DISA STIG scan... (this may take a few minutes)
Scan complete.
...
```

## Installation

Users install from GitHub:
```bash
pip install git+https://github.com/TheSTIGMan/stigman.git
```

Then run with:
```bash
sudo stigman
```

## Features

- Uses the official OpenSCAP scanner and ssg-ubuntu2404 STIG profiles.
- Chat UI to initiate scans, inspect issues, and execute bash-based remediations.
- Handles automated fallback for prerequisite checking and OS validation.
- Allows exporting HTML reports locally for later auditing.

## Security Warning

This application runs as `root` in order to execute DISA STIG scans and apply fixes using bash scripts generated by OpenSCAP. Review remediations carefully before approving them in the chat.
